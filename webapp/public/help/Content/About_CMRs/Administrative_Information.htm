Administrative Information

The Administrative Information tab lists:

  • some administrative details
  • the case or outbreak status
  • auditing or investigation information

Administrative details

The Administrative Information section of the page lists the following read-only information:

  • the record number for this event
  • the MMWR (CDC Morbidity and Mortality Weekly Report) year
  • the MMWR week
  • the date the event was created

Note: Depending on the disease you select, different questions appear on the CMR tabs. These questions are specific to the disease and appear once you have made the choice and selected one of the Save options. For more information about diseases, read Choosing a Disease and Editing a Disease.

Case/Outbreak

The Case/Outbreak section of the page records information about the status of this event in relationship to existing outbreaks.

 

Field Description
LHD case status Select a status from the list.
State case status Select a status from the list.
Outbreak associated Select a status from the list.
Outbreak Enter the name of the outbreak, if any. If an outbreak has been named, be sure to enter the name the same way every time you enter it for an event.

Auditing/Investigation

The Auditing Investigation section of the page is generally used by the local health department.

 

Field Description
Event name Enter the name of the event.
Jurisdiction of Residence The Jurisdiction of Residence field reflects the District assigned to the county that has been chosen in the County field on the Demographics tab.
Acuity Enter an acuity for this event.

 

When you click Save, the information is saved in the database.
