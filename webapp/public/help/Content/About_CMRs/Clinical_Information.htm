Clinical Information

The Clinical tab collects the following information about the disease and treatment. Enter as much information as you know.

  • Disease
  • Hospitalized Health Facilities
  • Status
  • Treatments
  • Clinicians
  • Diagnostic Facilities

Note: Depending on the disease you select, different questions appear on the CMR tabs. These questions are specific to the disease and appear once you have made the choice and selected one of the Save options. For more information about diseases, read Choosing a Disease and Editing a Disease.

 

Field Description
Disease Select the disease from the list. See Choosing a Disease for more information about what happens when you make this selection.
Onset date Enter the date of onset for the disease.
Date diagnosed Enter the date the person was diagnosed.
Hospitalized Select one of the options that describes if the person was hospitalized. Select Unknown if you do not know. If you select Unknown or No, you can skip the rest of the fields in this section and move to the Status section.

Add an additional entry section for hospital, if necessary.
Health facility Select a facility from the list only if the person was hospitalized.
Admission date Enter the date the person was admitted to a hospital.
Discharge date Enter the date the person was discharged from the hospital.
Medical Record Number Enter the person's medical record number.
Died Select one of the options from the list that indicates if the person died or not. Select Unknown if you do not know.
Date of Death Enter the date the person died, if any.
Pregnant Select one of the options from the list that indicates if the person is pregnant or not. Select Unknown if you do not know.
Pregnancy due date Enter the date the person is due to deliver.

Treatment given

 Select one of the options from the list that indicates if the person was treated or not. Select Unknown if you do not know.
Treatment

Enter a description of the treatment given to the person.

Add an additional entry section for treatment, if necessary.
Date of Treatment Enter the date the person was treated.
Date Treatment Stopped Enter the date the treatment stopped.
Clinician Last name Use the Search for existingfield, or enter the last name of the clinician in Last name.
Clinician First name

If you did not select the clinician's name from the list of existing names, enter the first name of the clinician.

Add an additional entry section for clinician, if necessary.
Clinician Middle name If you did not select the clinician's name from the list of existing names, enter the middle name of the clinician.
Phone type Select the type of the phone number for the clinician.
Area code Enter the area code for the clinician's phone number.
Phone number Enter the phone number for the clinician with or without the dashes.
Extension Enter the extension, if any, to select when calling the clinician.
Diagnostic Facility Name

Use the Search for existing field to find an existing Diagnostic Facility or type the name of the facility in Name.

Add an additional entry section for diagnostic facility, if necessary.
Diagnostic Facility Type Select the type of facility from the list.

 

When you click Save, the information is saved in the database.
