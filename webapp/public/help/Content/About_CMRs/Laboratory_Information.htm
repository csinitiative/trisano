Laboratory Information

The Laboratory tab collects information about the lab and the tests performed. Enter as much information as you know. The Lab and Test type fields are required.

Note: Depending on the disease you select, different questions appear on the CMR tabs. These questions are specific to the disease and appear once you have made the choice and selected one of the Save options. For more information about diseases, read Choosing a Disease and Editing a Disease.

 

Field Description
Lab Enter the name of the lab or search for an existing lab.
Test type Select the common test type from the list. The list of available common test types is limited by what is selected in the Disease field of the Clinical tab. If the list is too limited, select More Choices to see the entire available list, regardless of the chosen Disease.
Organism Select an organism from the list. The list of available common test types is limited by what is selected in the Disease field of the Clinical tab. If the list is too limited, select More Choices to see the entire available list, regardless of the chosen Disease.
Test result Select a result from the list.
Result value Enter the specific test result number.
Units Enter the units of the result number in the Result value field.
Reference range Enter the range, the test result had positive, negative, or healthy ranges.
Test status Select the test status from the list.
Specimen source Enter the closest type, such as blood, urine, and so on.
Collection date Enter the date the specimen was collected.
Lab test date Enter the date the test was performed.
Specimen sent to state Lab Select an answer from the list.
Comments Enter any relevant comments in the text field.

 

When you click Save the information is saved in the database.
