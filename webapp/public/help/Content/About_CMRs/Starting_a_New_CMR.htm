Starting a New CMR

You can start a new CMR after you have:

  • searched for a person and found that no information for the current occurrence of a disease or condition for that person exists in your database
  • clicked on Start a CMR with the criteria you searched on

After you click Start a CMR with the criteria you searched on, the New CMR page opens, displaying a collection of tabs. Use these tabs to enter information about the person and the event:

  • Demographic
  • Clinical
  • Laboratory
  • Contacts
  • Encounters
  • Epidemiological
  • Reporting
  • Investigation
  • Notes
  • Administrative

Disable/Enable tabs

The upper left corner of the page displays a link called Disable tabs. Click this to have all the tabbed pages displayed on one long scrolling page. The tabs at the top of the page disappear and all the information from them appears on a single page. Click Enable tabs to display the information on their tabbed pages again.

Hide/Show page sections

At many points through the forms, you will see a link called either Hide or Show.

  • Click Hide to hide the section of the page.
  • Click Show to show the section of the page.

Saving information

Two different buttons allow you to save your data and move on:

  • Click Save & Continue to save the information and continue working in edit mode on the current event.
  • Click Save & Exit to save the information, exit edit mode, and return to show mode for the current event.

Adding an entry section

On many tabs, there are sections that let you enter just one of a given item, such as Telephone or Email address on the Demographic tab; Hospital or Treatment on the Clinical tab; or many others on the other tabs as well. You may want to list more than one of these sections, if for example, the person has more than one phone number or has been treated at more than one hospital. Use the Add a... link to add another entry section. Each time you click Add a..., a new entry section is created.

Removing saved data

After you have saved your data, some of the entry sections contain a Remove check box. This makes it fairly simple to remove the data for that section. To remove one of these entry sections:

  1. Select the Remove check box.
  2. Click either Save & Continue or Save & Exit.

Searching for an existing entry

Many tabs contain search fields that let you enter part or all of a name and automatically perform a search for the text you enter. These fields search the database for entries that already exist, helping to prevent duplication. For example, the Clinical tab contains fields called:

  • Search for an existing clinician
  • Search for an existing diagnostic facility

To use any of these search fields:

  1. Begin typing in the field. Click Search if there is a Search button near the field.
  • If a match is found, a list of choices appears as a drop-down list. Select one of the existing entries. The fields in this section of the page display the stored information.
  • If a match is not found, move to the first entry field within this section of the page.
  1. Complete any other fields in the section of the page and continue.

Entering dates

Enter dates using one of the following methods:

  • Enter dates manually in one of the following formats. Use either a slash ( / ) or a dash ( - ) as a separator.
    • mm/dd/yy
    • mm/dd/yyyy
    • mm-dd-yy
    • mm-dd-yyyy
    • long form (January 1, 2009)
  • Enter dates using the pop-up calendar to select a date.

Back to list

At the top of the CMR page is a link called Back to list. Click this link to go to the Events list.

Show/Edit Modes

Many of the tabs have both a Show mode and an Edit mode:

  • In Show mode, you can view data, but not edit it. On some of the tabs, there are additional command links in red text along the top to provide additional features.
  • In Edit mode, you can edit the data displayed.
