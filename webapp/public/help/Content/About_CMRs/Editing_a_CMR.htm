Editing a CMR

After you have created a CMR, you may need to change some of the information on it. To edit an existing CMR:

  1. Search for the CMR using either the person's information or the event information.
  2. Select the event you want to edit from the list of search results. A page of tabbed information opens.
  3. Click Edit at the top left of the page. The fields on the page become available for editing.
  4. Make any necessary changes and click Save & Continue to save your changes keep editing or Save & Exit to save your work and return to the search results.
