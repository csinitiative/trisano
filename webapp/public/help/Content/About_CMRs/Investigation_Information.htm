Investigation Information

The Investigation tab is the default location for any custom forms that have been created using Form Builder. Form Builder is a tool that allows you to extend the capabilities of CSI TriSano™ by capturing additional information for existing or new public health situations. It can be extremely useful during a disease outbreak. It helps you respond quickly to emerging or changing health events by allowing you to create customized forms. The new form can help you capture additional information for each of the event types including:

  • Morbidity events (CMRs)
  • Contact events
  • Encounter events
  • Place Exposures

Disease-specific forms are assigned to an event automatically when you select a disease on the Clinical tab and save the event. If you change the disease for an event or design a new form using Form Builder, you can add or remove forms from an event using the following instructions.

To add a form to this event:

  1. Click Add/Remove Forms for this event. The page displays:
  • Forms in use
  • Forms available for use
  1. In the Forms available for use section of the page, select the Add to event check box at the end of the row for a form you want to add and click Add Forms. The form becomes part of the Investigation tab.

To remove a form:

  1. Click Add/Remove forms for this event. A page opens listing:
  • Forms in use
  • Forms available for use
  1. In the Forms in use section of the page, select the Remove from event check box at the end of the row for the form you want to remove and click Remove forms.
  1. Click OK to confirm that you want to remove the form or click Cancel to keep the form.
