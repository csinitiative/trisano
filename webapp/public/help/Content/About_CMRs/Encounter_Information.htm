Encounter Information

An encounter is a meeting between clinical staff and a patient. Use the Encounter tab to record information about that patient meeting. Enter as much information as possible.

 

Field Description
Investigator Select the name of the investigator from the list.
Encounter date Enter the date of the encounter.
Location Select a location from the list.
Description Enter a description for the encounter. Give as much detail as possible.

When you click Save & Continue, the encounter is recorded and in addition to being shown on the top of the Encounter tab, it is visible at the bottom of the tab. As you record more encounters, they are added to the list.

To add more information about the encounter, click Edit Encounter.

In addition to the basic encounter information displayed, the following tabbed pages are added. Explanations for these fields are the same as for the tabbed pages for the original patient:

  • Clinical
  • Laboratory
  • Investigation
  • Notes

The fields on these pages are the same as those for the original patient. Click the links in the bulleted list for explanations of the fields.
