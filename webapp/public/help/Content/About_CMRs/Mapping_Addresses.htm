Mapping Addresses

The mapping feature allows you to display a map that contains a tag showing the location of the address. This allows you to see in a graphical way, the precise location of the patient, contact, or place exposure.

You can map an address from several places in the application:

  • Demographic tab of all event types
  • Contact events on the Contacts tab of Morbidity events
  • Place Exposures on the Epidemiological tab of Morbidity events

The general instructions for mapping addresses follow:

  1. Complete as much of the address information as you can.
  2. Click Lookup Address.
  3. Click Show Map.

For detailed instructions for each address type, view the topic for that page.
