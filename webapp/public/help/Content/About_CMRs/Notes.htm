Notes

The Notes tab brings together assorted information for an event. It lists:

  • Attachments that exist for this event, if any.
  • Earlier notes about this event that may have already been created. When the event is created a note is added for tracking purposes. Click All, Administrative, or Clinical to view different groups of notes.
  • Tasks associated with this event.

To add a note:

  1. While in Edit mode, enter the text for the note in the New note field.
  2. If the note is an administrative note, select the Is Admin check box.
  3. Click Save. The note is added to the Earlier notes section of the page. It lists:
  • the user name of the person who entered the note
  • the type of note: Administrative or Clinical
  • the date and time the note was saved
  • the jurisdiction, if the note is for an event

Note: Saved notes are permanent. They cannot be deleted. Use the strike-through feature to indicate that the note is no longer valid.

To strike out a note:

  1. While in Edit mode, select the Strike Through check box.
  2. Click Save.
