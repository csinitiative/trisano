Demographic Information

The Demographic tab collects the following information for a person:

  • Name
  • Address
  • Age
  • Contact information
  • Gender, ethnicity, race, and language

Enter as much information as you can. Last Name is the only field that is required.

Note: Depending on the disease you select, different questions appear on the CMR tabs. These questions are specific to the disease and appear once you have made the choice and selected one of the Save options. For more information about diseases, read Choosing a Disease and Editing a Disease.

 

Field Description
Last name Enter the last name of the person.
First name Enter the first name of the person
Middle name Enter the middle name of the person.
Parent/Guardian Enter the name of the person's parent or guardian.
Street number Enter only the number part of the person's street address.
Street name Enter the name of the street where the person lives. Be sure to spell out the address completely. For example, use 12 West 52 Street South and not 12 W. 52 St. S. If the address is a PO Box, enter the PO Box and number in Street name and leave Street number blank.
Unit number Enter the unit, suite, or apartment number of the apartment unit, multi-family housing complex, or office building, if any.
City Enter the city where the person lives. Spell out the entire name. For example, use Los Angeles, not LA.
State Select the state where the person lives.
County Select the county where the person lives.
Zip Code Enter the Zip code for the person's address.
Latitude Click Look Up Address, then Update Coordinates to automatically complete this field. The number that appears is the geographical latitude of the address described for this patient.
Longitude Click Look Up Address then Update Coordinates to automatically complete this field. The number that appears is the geographical longitude of the address described for this patient.
Date of Birth Enter the date the person was born.
Age Enter the age of the person.
Phone type Select the type of the phone number supplied by the person.
Area code Enter the area code for the person's phone number.
Phone number Enter the phone number for the person. You can enter the number with or without the dash.
Extension Enter the extension, if any, to select when calling the person. Add an additional entry for telephone, if necessary.
Email address Enter the email address for the person. Add an additional entry for email, if necessary.
Birth gender Select the gender the person was at birth.
Ethnicity Select the ethnicity of the person, from among those listed.
Race Select the race of the person, from among those listed. You can select more than one race by holding down the Ctrl key and making additional selections.
Primary language Select the language of the person, from among those listed.

 

When you click Save the information is saved in the database.

Mapping Addresses

After you have entered and saved address information for a patient, you may see a link at the bottom of the Address area called Lookup Address. This feature allows you to check the address of this patient for validity. In addition, once you have checked the address, you can show it on a map. Refer to Mapping Addresses on the Demographic Tab  for more information.
