Deleting a Morbidity Event

To delete a morbidity event:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the event you want to delete.
  3. Click Delete. The page displays a confirmation box.
  4. Click Yes to confirm deletion or No to leave the event as is.

If you confirm deletion of the event:

  • it remains in the database
  • it appears as dimmed in the event list
  • it can still be viewed using the Show command
  • it is included in searches
  • it appears as striked out in search results
  • it is not exported to the CDC or to an IBIS query

In addition, you can still use deleted events for the following:

  • you can edit deleted events
  • you can route deleted events
  • you can copy a deleted event to a new event, where it will not appear as deleted
