Verifying Addresses

Most events contain address information. The address verification feature allows you to verify address information based on the entries made in the combination of all the address-related fields. It compares them with a database of all known addresses and displays updated, verified information for you, if you have entered enough information.

The address verification feature is available in the following places:

  • Demographic tab of all event types
  • Contact events on the Contacts tab of Morbidity events
  • Place Exposures on the Epidemiological tab of Morbidity events

To use address verification:

  1. Open one of the above-listed pages for the patient, contact, or place.
  2. Locate the Address fields and make sure they are as complete as possible. The address look-up feature is fairly forgiving, so you don't need to complete every field if you don't have all the information. In fact, you can use the look-up feature to help complete missing address information.
  3. Click Look Up Address. The application checks the address and if there is sufficient address information, displays the verified (sometimes corrected) address below the Look Up Address link. If there isn't enough address information, the application tells you so. In this case, you will not be able to verify the address nor will you be able to include this patient, contact, or place in mapped results.
  4. If the look-up feature found the address you entered, you can:
  • Click Update Coordinates. The Longitude and Latitude fields will be updated, but the Address fields will remain as you entered them.
  • Click Update Address. The Longitude and Latitude fields will be updated and the Address fields will be updated with the found address information.
  1. Click Save & Continue.
