Reporting Information

Reporting information is data about the source of the report, the type of agency, the person making the report, and the dates of the report. Use the Reporting tab to enter as much information as possible about the source of the report.

Note: Depending on the disease you select, different questions appear on the CMR tabs. These questions are specific to the disease and appear once you have made the choice and selected one of the Save options. For more information about diseases, read Choosing a Disease and Editing a Disease.

 

Field Description

Reporting Agency Name

Use the Search for a Reporting Agency field to find an existing reporting agency or if none is found, enter the name of the agency in Reporting Agency Name.

Click Add a new reporting agency to change the Reporting Agency Name.

 
Area code Enter the area code for the agency.
Phone number Enter the phone number for the agency. You can enter the number with or without the dash.
Extension Enter the extension, if any, to select when calling the agency.
Types Select the type of reporting agency from the list.
Reporter last name

Enter the last name of the reporter.
Reporter first name Enter the first name of the reporter.
Area code Enter the area code for the reporter.
Phone number Enter the phone number for the reporter. You can enter the number with or without the dash.
Extension Enter the extension, if any, to select when calling the reporter.
Results reported to clinician date Enter the date the results were reported to a clinician.
Date first reported to public health Enter the date this was first reported to a public health authority.

 

When you click Save the information is saved in the database.
