Core Data vs. Disease-Specific Data

CSI TriSano™ is an extremely flexible tool for gathering and reporting public health surveillance data. Although you may not notice it while you are entering patient information, there are two types of data that you will record:

 

  • Core data consists of all the built-in fields available in the application. These fields will appear for every disease.
  • Disease-specific data consists of fields defined by a user in the creation of forms for a specific disease. Disease-specific data differs from disease to disease and can appear nearly anywhere, on any tab, as defined when you create forms for the disease.
