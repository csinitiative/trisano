Mapping Addresses on the Demographics Tab

The mapping feature allows you to display a map that contains tags showing the location of the patient's address. This allows you to see the location of the patient's address in a graphical way.

 

To map an address:

  1. Click Events in the upper right corner of the page.
  2. Locate the patient whose address you want to map and click either Show or Edit. You do not have to be in Edit Mode to map the address for the patient.
  3. Select the Demographics tab. Locate the Address for the patient as in the following illustration.

 

 

  1. Click Look Up Address.
  2. Click Show Map. A map opens showing the location of the patient just below the address as shown in the following illustration. The red tag indicates the address of the patient.
