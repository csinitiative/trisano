What is the difference between core data and disease specific data?

Although you may not notice it right away while you are entering patient information, there are two types of data that you will work with:

 

  • Core data consists of all the built-in fields available in the application. These are static, uneditable fields that are not dependent on the chosen disease.
  • Disease-specific data consists of fields defined by a user in the creation of forms for a specific disease. Disease-specific data differs from disease to disease and can appear nearly anywhere, on any tab, as defined when you create forms for the disease.
