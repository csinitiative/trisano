How do I search for people or events?

To perform a search:

  1. On the Search page, enter only the fields you want to search on. The more information you enter, the more narrow your search. Use the descriptions below to determine what you want to enter.
  2. After you have entered the criteria for the search, click Submit Query. Results are reported at the bottom of the page.
  3. Click the Record Number of the event you want to view. The page for that event opens in Show mode.
  4. Click Start Over to clear the search criteria from the page and start over.

Name Criteria

Enter as much of the person's name as you know.

  • Use the Full-text name field to type the person's full name, first and last.
  • Use the Starts-with first name field to enter the person's first name.
  • Use the Starts-with last name field to enter the person's last name.

If you enter text in the Starts-with first name or Starts-with last name fields, the search ignores anything you may have entered in the Full-text name field.

Demographic Criteria

Enter any of the following Demographic information for the events you want to find:

  • City
  • County
  • Gender
  • Date or year of birth

Clinical Criteria

Enter any of the following Clinical information for the events you want to find:

  • Select one or more Diseases.
  • Select a status from the Pregnant list.

Event Criteria

  • Select an Event Status from the list.
  • Select an Event Type from the list.
  • Select a status from the Sent to CDC list.
  • Enter dates in the Entered-on date range fields.
  • Enter the Record Number, if you know it.
  • Select a Jurisdiction of Investigation from the list.
  • Select a State Case Status from the list.
  • Select an LHD Case Status from the list.
  • Select an investigator in the Investigated by field.

Epidemiological and Reporting Criteria

  • Use Other Data 1 and Other Data 2 to search fields with the same names on the Epidemiological tab of existing events.
  • Enter a range of dates in First reported to public health. These dates search the field with the same name on the Reporting tab of existing events.
