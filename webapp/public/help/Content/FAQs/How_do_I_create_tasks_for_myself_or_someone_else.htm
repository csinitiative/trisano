How do I create tasks for myself or someone else?

A task is some form of follow-up action that must be undertaken for the patient or a contact. Tasks typically fall into three categories: appointments, call backs, and treatments. You can create tasks only for Morbidity or Contact events. To add a task:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the Morbidity or Contact event you want to create a task for.
  3. Click Show.
  4. Click Add task. The New Task page opens.
  5. Specify settings as explained in the following table.
  6. Click Create. The task will be listed at the top of the page and will be added to the Notes tab.

 

Field Description
Name Enter a name for the task.
Description Enter a description for the task.
Category Select a category from the list.
Priority Select a priority from the list.
Due date Enter a due date for the task.
Repeat Select a repeat interval from the list
Until Enter the date the task should end.
User Select a user from the list.
