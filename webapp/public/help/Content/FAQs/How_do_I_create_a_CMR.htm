How do I create a CMR?

A confidential morbidity report (CMR) is a collection of information that describes an encounter with a patient that must be reported to a public health department. The report includes details that public health authorities use in order to track disease occurrences and analyze patterns.

 

To create a CMR:

  1. Click New CMR in the upper right corner of the page.

    The search page opens. Before you can create a new CMR, you must first search for the person associated with it to be sure that neither the person nor the event has already been created.
  2. Type the name of the person and birth date, if you know it. You can search on only one of the fields, if that is all the information you have.
  3. Select Use starts with search to have the search match the characters you enter exactly. For example, if you enter "smi" the results would reveal "Smith" but not "Smythe." If you leave the box unchecked, search performs a more fuzzy search and returns events that match closely, but not precisely.
  4. Click Search.

    A list of names that closely resembles the name you entered appears. Or, if there are no names that are even close, no names appear and a message tells you that there were no matches.



    If you see the name of the person for this event, you can do one of the following:
  • Edit an existing event for this person.
  • Show an existing event for this person.
  • Create and edit a CMR for this person.

If you do not see the name of the person for this event or the message tells you that there were no matches, click Start a CMR with the criteria you searched on.
