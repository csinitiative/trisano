Defining and Editing Short Names

To define a new Short Name or edit an existing one:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Configure System for CSV Export. The CSV Export Column List opens displaying one field per row.
  3. Click Edit. A text field opens allowing you to edit an existing Short Name or enter a new one. Short Names can contain up to ten characters.

  1. Edit the existing name, if there is one, or enter a new one.
  2. Click OK to save the new name or Cancel to dismiss your changes.
