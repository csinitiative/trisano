CSV Export Column Details

The CSV Export Column details page lists the Long Names for the fields from the morbidity event pages and the Short Name that has been defined for each. You can use the list to define and edit Short Names, which are easier to use in a CSV file. To view the CSV Export Column List:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Configure System for CSV Export. The CSV Export Column List opens displaying one field per row.

Click Edit to define a new Short Name or edit an existing one.
