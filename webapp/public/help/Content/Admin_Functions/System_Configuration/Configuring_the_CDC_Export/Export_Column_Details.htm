Export Column Details

The Export Column List displays information that is included in CDC report. You can use the list to show, edit, delete, and create export columns. To view the Export Column List:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Configure System for CDC Export. The Export Column List opens displaying one column per row.

For each column, the list shows:

  • the name of the event queue as it has been defined
  • the jurisdiction for the queue

Click on the commands in red text to perform an action on the export column.
