Editing an Export Column

Edit Mode lets you edit the information for the export column. To edit an export column:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Configure System for CDC Export. The Export Column List opens displaying one column per row.
  3. Click Edit in the row of the column you want to edit.
  4. Make any changes to the information for this export column.
  5. Click Update to save the new information. To return to the column list without saving your changes, click Back to Export Columns.
