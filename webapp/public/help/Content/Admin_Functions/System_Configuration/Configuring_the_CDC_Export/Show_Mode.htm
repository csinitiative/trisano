Show Mode

Show Mode lets you see all the information for an export column, but does not allow editing. The following illustration shows export column information in show mode.

Show mode also displays the acceptable values for the columns for certain datatypes, such as radio buttons or check boxes, at the bottom of the page. In show mode, you can add a new valid value, edit an existing value, or delete an existing value. 

 

To return to the list of export columns, click Back to Export Columns.

 

To edit this export column, click Edit.
