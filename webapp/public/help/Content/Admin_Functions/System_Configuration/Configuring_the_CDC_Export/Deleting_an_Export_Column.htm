Deleting an Export Column

To delete an export column:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Configure System for CDC Export. The Export Column List opens displaying one column per row.
  3. Click Delete in the row of the column you want to delete. The row is deleted and the page refreshes without the export column listed.
