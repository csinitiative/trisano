Creating a New Export Column

To create a new export column:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Configure System for CDC Export under System Configuration. The Export Columns page opens.
  3. Click Create New Export Column. A blank export column page opens.
  4. Complete the export column fields as described in the table below.
  5. Click Create. The new export column is added to the list.

 

Field Description
Associated diseases Select the diseases associated with the export column data.
Descriptive name Enter a descriptive name for the export data. This name is the first field listed on the Export Columns list page.
CDC Column Name Enter the name of the column as required by the CDC.
Disease Group Select the disease group that this column is related to.
Data Source Select from the list the source of the data. Possible sources are: Core Data, Formbuilder Data, System Data.
Data Type Select from the list the type of data used by this field.
Table Name Enter the Table Name for this column.
Column Name Enter the Column Name for this column.
Required Select Yes or No, depending on whether or not the information is required.
Start Position Enter the starting position for this column, as indicated by the CDC requirements.
Length to Output Enter the column width, as indicated by the CDC requirements.
