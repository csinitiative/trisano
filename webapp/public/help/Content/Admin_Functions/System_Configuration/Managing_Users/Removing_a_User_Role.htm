Removing a User Role

You can remove a role from a user's profile to change the user's access. To remove a role:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Manage Users under System Configuration. The Users page opens listing the existing users.
  3. Click Edit in the row of the user you want to edit.
  4. Scroll down to the Edit Roles section of the page.
  5. Click Remove next to the role you want to remove from this user's profile.
  6. Click Update to save the changes you just made.
