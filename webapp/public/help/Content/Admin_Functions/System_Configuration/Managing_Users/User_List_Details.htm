User List Details

The User List displays all the users that have been defined. You can use the list to show, edit, and create user identities for use in CSI TriSano™. To view the User List:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage Users. The User List opens displaying one user per row.

For each user, the list shows:

  • the user's full name (first and last names), if they have been entered
  • the user ID
  • the username

Click on the commands in red text to perform an action on a user.
