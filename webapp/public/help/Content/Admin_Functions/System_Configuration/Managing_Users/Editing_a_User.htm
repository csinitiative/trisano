Editing a User

Edit Mode lets you edit the information and roles for the user. To edit a user:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage Users. The User List opens displaying one user per row.
  3. Click Edit in the row of the user you want to edit.
  4. Make any changes to the information for this user.
  5. Click Update to save the new information. To return to the list of users without saving your changes, click Back to Users.
