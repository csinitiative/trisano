Creating a New User

Before creating a new user, consider what type of user this will be and what role this user will have. You can create a new user only if you have the correct permissions to do so. To create a new user:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Manage Users under System Configuration. The Users page opens listing the existing users.
  3. Click Create New User. A blank user page opens.
  4. Complete the user fields as described in the following table.

 

Field Description
UID Enter a user ID for the new user.
User name Enter the full name of the user.
First name Enter the first name of the user.
Last name Enter the last name of the user.
Given name Enter the given name of the user. A person's given name is the name given to him or her at birth, the person's formal first name.
Generational qualifier Enter a generational qualifier for the user, if there is one.
Initials Enter the user's initials.

 

  1. Add Roles for this user as follows:
    1. Click Add Role.
    2. Select a Jurisdiction from the list.
    3. Select a Role for this user from the list.
  2. Click Create. The new user is added to the list of users.
