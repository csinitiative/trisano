About the CSV Export

CSI TriSano™ allows you to export a list of fields and their contents available on the morbidity event pages. This export creates a comma separated value (CSV) file that you can then view in a spreadsheet program.

When you choose Export All to CSV on the Event List page, you can choose the option to Use Short Column Names for the fields instead of the long names. The Configure System for CSV Export area lets you define the short names.

The CSV export is a simple list; more advanced reporting is available using the AVR Tools.
