Creating System Messages

System messages are brief notices that are displayed in the top banner area of the CSI TriSano™ application. You can use these messages to broadcast notices to all users.

To create a system message:

  1. Click Admin in the upper right corner of the page.
  2. Select Send System Messages under System Configuration.
  3. Click Create New System Message.

 

 

  1. Select a date and time for the message to expire or leave all fields blank to create a permanent message.
  2. Enter the message in the Message field.
  3. Click Create. The application reports that the message was successfully created.

 

 

Working with Messages

After you have created a message it will be listed on the System Messages page. You can perform the following tasks on messages:

  • Click Show All Messages to display all messages that are current or that have expired.
  • Click Expire Now to remove the current message from the banner.
  • Click Show to display the message its field in show mode.
