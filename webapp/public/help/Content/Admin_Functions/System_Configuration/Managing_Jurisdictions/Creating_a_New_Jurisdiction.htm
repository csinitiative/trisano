Creating a New Jurisdiction

To create a new jurisdiction:

  1. Click Admin in the upper right corner of the page.
  2. Select Manage Jurisdictions under System Configuration. The jurisdiction list opens, displaying one jurisdiction per row.
  3. Click Create new jurisdiction. The following page opens allowing you to define a jurisdiction.

 

 

  1. Enter the name of the jurisdiction in the Name field.
  2. Enter a short name for the jurisdiction in the Short name field.
  3. Click Create. The application displays a message telling you that the jurisdiction was successfully created.

 

To add counties to a jurisdiction, see Adding Counties to a Jurisdiction.
