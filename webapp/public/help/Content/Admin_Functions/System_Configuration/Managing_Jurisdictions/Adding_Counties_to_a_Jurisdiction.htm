Adding Counties to a Jurisdiction

Public health jurisdictions generally encompass a number of counties. Counties can be added to a jurisdiction using the System Codes area of the application. To add counties to a jurisdiction:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Manage System Codes under System Configuration. The Codes page opens listing the existing codes.
  3. Locate the County Code and click on the word County or click List. The list of County Codes opens.
  4. To assign a county to a jurisdiction, locate a county that belongs to the jurisdiction in question.
  5. Click Edit in the row for that County.
  6. Select the Jurisdiction that this county belongs to.
  7. Click Update. The application displays a message telling you that the code was successfully updated.
