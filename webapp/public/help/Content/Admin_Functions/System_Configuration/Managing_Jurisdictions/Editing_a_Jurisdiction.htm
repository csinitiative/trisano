Editing a Jurisdiction

To edit a jurisdiction:

  1. Click Admin in the upper right corner of the page.
  2. Select Manage Jurisdictions under System Configuration. The Jurisdiction list opens, displaying one jurisdiction per row.
  3. Review the list and locate the jurisdiction you want to edit.
  4. Click Edit. The page displays the jurisdiction in edit mode.
  5. Make any necessary changes and click Update to save your changes. The application reports that the Jurisdiction was successfully saved.
  6. Click Back to Jurisdictions to return to the Jurisdiction list.
