Showing a Jurisdiction

To show a jurisdiction:

  1. Click Admin in the upper right corner of the page.
  2. Select Manage Jurisdictions under System Configuration. The Jurisdiction list opens, displaying one jurisdiction per row.
  3. Review the list and locate the jurisdiction you want to view.
  4. Click View. The page displays the jurisdiction in show mode. You cannot edit the jurisdiction in show mode.
  5. To edit the jurisdiction, click Edit. Make any necessary changes and click Update to save your changes. The application reports that the jurisdiction was successfully saved.
  6. Click Back to Jurisdictions to return to the Jurisdiction list.
