Jurisdiction List Details

The Jurisdiction list displays in alphabetical order all jurisdictions that have been defined. To view the Jurisdiction list:

  1. Click Admin in the upper right corner of the page.
  2. Select Manage Jurisdictions under System Configuration. The list opens, displaying one jurisdiction per row. Each jurisdiction listed shows:
  • jurisdiction name
  • jurisdiction short name

Click on the commands in red text to perform an action on a jurisdiction.
