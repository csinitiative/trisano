Editing Help Text For Core Fields

To edit existing help text for core fields:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Add Help Text to Core Fields under System Configuration. The Core Fields page opens listing CSI TriSano™ core fields.
  3. Locate the row that contains the core field whose help text you want to edit.
  4. Click Edit. The Edit Core Field page opens.
  5. Make your changes to the text in the Help Text entry box.
  6. Click Update when you are done editing. The help text is saved and you are returned to the core field in Show mode.
