About Help Text for Core Fields

CSI TriSano™ collects a large amount of data about patients and diseases. There are numerous fields that users complete during data collection. You can add help text to core fields to help users understand the purpose of a field and what sort of entry is required.

When you define help text, a small question mark appears next to the field to let users know that there is additional information available to help them.

 

 

When the user hovers the cursor over the question mark, the help text defined for the field appears on the screen.
