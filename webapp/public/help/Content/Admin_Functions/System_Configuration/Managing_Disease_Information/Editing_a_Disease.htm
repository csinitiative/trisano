Editing a Disease

Edit Mode lets you edit the information for the disease. To edit a disease:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage Diseases. The Disease List opens displaying one disease per row.
  3. Click Edit in the row of the disease you want to edit.
  4. Make any changes to the information for this disease.
  5. Click Update to save the new information. To return to the list of diseases without saving your changes, click Back to Diseases.
