Disease List Details

The Disease List displays all the diseases that have been defined. You can use the list to show, edit, and create new diseases for use in CSI TriSano™. To view the Disease List:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage Disease Information. The Disease List opens displaying one disease per row.

For each disease, the list shows:

  • the full name of the disease
  • its status (whether or not the disease is active)
  • its CDC code, if there is one

Click on the commands in red text to perform an action on a disease.
