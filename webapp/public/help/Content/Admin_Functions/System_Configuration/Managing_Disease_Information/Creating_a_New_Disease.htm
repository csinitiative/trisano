Creating a New Disease

To create a new disease:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Manage Disease Information under System Configuration. The Diseases page opens listing the existing diseases.
  3. Click Create New Disease. A blank disease page opens.
  4. Complete the disease fields as described in the table below.
  5. Click Create. The new disease is added to the list of diseases.

 

Field Description
Disease name Enter the name of the disease.
Active Select whether or not this disease is currently active. If a disease is active, it is listed and can be selected on the Disease list on the Clinical tab of a new CMR. If a disease is not active, it does not appear on the list. However, if the disease was active and selected in the past, it remains as part of CMRs created in the past.
Contact lead-in question The contact lead-in question appears on the Contact tab of a CMR. It helps users ask the right questions and collect useful information.
Place lead-in question The place lead-in question appears in the Place Exposures section of the Epidemiological tab of a CMR. It helps users ask the right questions and collect useful information.
Treatment lead-in question The treatment lead-in question appears in the Treatments section of the Clinical tab of a CMR. It helps users ask the right questions and collect useful information.
CDC/NETSS code Enter the Centers for Disease Control and Prevention/National Electronic Telecommunications System for Surveillance code.
CDC/NETTS export disease when case is: The CDC publishes a list of diseases that they are collecting data on and the statuses that determine when they want to learn about them. These criteria indicate when a disease should be reported to the CDC electronically. Select the criteria for reporting.
AVR Group Select the AVR Groups for this disease. The disease can be part of one or more AVR Groups.
