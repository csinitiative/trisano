Event Queue Details

The Event Queue List displays all the event routing queues that have been defined in the database. You can use the list to show, edit, delete, and create event queues for use in CSI TriSano™. To view the Event Queue List:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage Event Routing Queues. The Event Queue List opens displaying one queue per row.

For each queue, the list shows:

  • the name of the event queue as it has been defined
  • the jurisdiction for the queue

Click on the commands in red text to perform an action on the event queue.
