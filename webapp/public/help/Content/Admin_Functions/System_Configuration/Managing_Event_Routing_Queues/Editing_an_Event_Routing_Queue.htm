Editing an Event Routing Queue

To edit an event routing queue:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Manage Event Routing Queues under System Configuration. The Event Queues page opens listing the existing queues.
  3. Locate the queue you want to edit and click Edit on the same row as that queue. The event queue page opens.
  4. Edit the name for the queue, if necessary.
  5. Edit the jurisdiction, if necessary.
  6. Click Update. The queue is saved with the changes and displayed on the page.
  7. Click Back to Event Queues to return to the queue list.
