Listing Code Elements

The elements in a code list are the actual items that will appear in the application as selections in a field. Although you cannot define the codes themselves, you can define elements.

 

To list the elements for a code:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage System Codes. The System Code List opens displaying one code type per row.
  3. Locate the code type you want to work with and either click on the code type name or click List. The page that opens lists all of the code elements that have been defined for this type. From here you can define, show, edit, or delete code elements.
