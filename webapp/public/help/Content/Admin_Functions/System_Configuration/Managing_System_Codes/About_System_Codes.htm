About System Codes

System codes are abbreviations for selections used in many places in the application. Codes are organized in groups that place them together as selections for a field.

 

For example, you may want to populate a drop-down field located on a form with the names of the counties in your state. You could use the System Codes to enter both the abbreviation for a county as well as the full name. You could select County from the list of code types and add all the counties you need.

 

The list of code types is pre-determined, but you can add and edit the elements included for each code as necessary.
