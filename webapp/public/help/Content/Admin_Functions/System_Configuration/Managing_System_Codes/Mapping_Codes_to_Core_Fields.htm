Mapping Codes to Core Fields

The pre-defined system codes are automatically mapped to core fields according to the following table.

 

Code Group Core Field
age_type Used only for the CDC Export.
case LHD Case Status and State Case Status on the Administrative tab.
contactdispositiontype Disposition on the Contacts tab.
contact_type Contact type on the Contacts tab.
county County on the Demographic tab.
ethnicity Ethnicity on the Demographic tab.
gender Birth gender on the Demographic tab.
imported Imported from on the Epidemiological tab.
lab_interpretation Interpretation on the Laboratory tab.
lab_test_types Test type on the Laboratory tab.
language Language on the Demographic tab.
location Phone type anywhere on a CMR.
race Race on the Demographic tab.
specimen Specimen source on the Laboratory tab.
state State anywhere on a CMR.
task category Category on an Event Task
telephonelocationtype Phone type anywhere on a CMR.
yesno Any Yes/No type drop-down on a CMR.
