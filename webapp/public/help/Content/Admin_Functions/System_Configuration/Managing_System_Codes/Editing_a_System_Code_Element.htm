Editing a System Code Element

Edit Mode lets you edit the details for a code element. To edit a system code element:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage System Codes. The System Code List opens displaying one code type per row.
  3. Locate the code type you want to work with and either click on the code type name or click List.
  4. Click Edit in the row of the code element you want to edit.
  5. Make any changes to the information for this element.
  6. Click Update to save the new information. To return to the list of codes without saving your changes, click Back to Codes.
