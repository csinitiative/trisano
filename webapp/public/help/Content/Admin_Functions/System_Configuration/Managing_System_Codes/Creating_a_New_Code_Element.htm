Creating a System Code Element

System codes are abbreviations for selections used in many places in the application. Codes are organized in groups that place them together as selections for a field. The list of code types is pre-determined, but you can add and edit the elements included for each code as necessary.

 

To define a System Code Element:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage System Codes. The System Code List opens displaying one code type per row.
  3. Locate the code type you want to work with and either click on the code type name or click List.
  4. Click the Create button that appears in the upper right area of the page.
  5. Complete the fields as described in the following table.
  6. Click Create. The application displays a message telling you that the code was successfully created.

 

Field Description
Code Enter the code that you want to appear as a selection.
Description Enter a description for the code so that you know what it means.
Sort Order Enter a number to specify where on the list of choices, this particular item should appear. It is a good idea to leave some space between Sort Order numbers so that you can add other items to the list at a later date.
