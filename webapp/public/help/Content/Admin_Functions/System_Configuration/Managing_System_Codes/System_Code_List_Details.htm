System Code List Details

The System Code List displays all the codes that have been defined. You can use the list to access each pre-defined code and then show, edit, and delete code member elements for use in CSI TriSano™. To view the System Code List:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage System Codes. The System Code List opens displaying one code per row. For each code, the list shows:
    • the system code
    • the description of that code
