Editing a Role

To edit a role:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage Roles. The Role List opens displaying one Role per row.
  3. Click Edit in the row of the role you want to edit.
  4. Make any changes to the privileges for this role.
  5. Click Update to save the new information. To return to the list of roles without saving your changes, click Back to Roles.
