Deleting a Role

You can delete a role:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Manage Roles under System Configuration. The Roles page opens listing the existing roles.
  3. Click Delete in the row of the role you want to delete. The application asks you to confirm that you want to delete this role.
  4. Click Yes to delete the role or No to keep it.
