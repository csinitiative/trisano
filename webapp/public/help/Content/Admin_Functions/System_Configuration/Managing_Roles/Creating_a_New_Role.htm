Creating a New Role

Before creating a new role, consider what type of role this will be and what privileges this role will have. You can create a new role only if you have the correct privileges to do so. To create a new role:

  1. Click the Admin link in the upper right corner of the page. The Admin page opens.
  2. Click Manage Roles under System Configuration. The Role List page opens listing the existing roles.
  3. Click Create new role. A blank role page opens.
  4. Enter a name in the Role name field.
  5. Add as many privileges for this role as necessary by clicking Add Privilege and selecting a Privilege from the list.
  6. Click Create. The new role is added to the list of roles and the application returns you to the Roles List.
