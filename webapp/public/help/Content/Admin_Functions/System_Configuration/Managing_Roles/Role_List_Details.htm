Role List Details

The Role List displays all the roles that have been defined in CSI TriSano™. You can use the list to show, edit, delete, and create roles for users. To view the Role List:

  1. Click Admin in the upper right corner of the page.
  2. Under System Configuration, click Manage Roles. The Roles List opens displaying one role per row.

Click on the commands in red text to perform an action on a role.
