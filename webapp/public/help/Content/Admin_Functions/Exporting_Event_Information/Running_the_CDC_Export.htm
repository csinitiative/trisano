Running the CDC Export

The CDC Export function generates what's called an MMWR report to be sent to the CDC. This is the short name for the Morbidity and Mortality Weekly Report. The report lists all the cases stored in your database of selected nationally notifiable diseases that the CDC requires in a weekly report.

 

To generate the report:

  1. Click Admin in the upper right corner of the page.
  2. Under Export Event Information, click CDC Export. The export page opens. It offers you three different reports:
  • the weekly CDC report — which includes data from the current week as well as the previous week.
  • the MMWR weeks emcompassed by the dates you enter — which includes data from only the weeks whose dates you enter in the Start Date and End Date fields. The Start and End dates include all reportable records from the start of the MMWR week in which the Start Date field falls and the end of the MMWR week in which the End Date field falls.
  • the annual CDC summary report for the year — which includes data from the entire year you enter in the year field.

To generate the weekly CDC report:

  1. Click The weekly CDC report. A dialog opens prompting you to save the report.
  2. Click Save and specify where you want to save the report. The report is named for the week numbers it represents.
  3. Click Save once again to finish saving the report.

To generate the report for the MMWR weeks encompassed by the dates you enter:

  1. Enter dates in both the Start Date and End Date fields. You must enter dates in the format YYYY-MM-DD. No other format is accepted in these fields.
    Note: Weekly reports may not span MMWR years. If you enter dates that span MMWR years (e.g., 2008-12-15 through 2009-01-15) the start date will be changed to MMWR week 1 of the end date's year.
  2. Click Submit.
  3. Click Save and specify where you want to save the report. The report is named for the week numbers it represents.
  4. Click Save once again to finish saving the report.

To generate the annual CDC summary report for the year:

  1. Enter the year in the Year field.
  2. Click Submit.
  3. Click Save and specify where you want to save the report. The report is named for the year it represents.
  4. Click Save once again to finish saving the report.
