Merging People

Duplicate records can sometimes be created when anyone entering data fails to recognize that the person for whom they are creating a record already exists in the database. The merge people feature allows you to conveniently combine duplicate records.

Note: Use the merge feature with caution. It is a powerful tool that combines records for people and permanently deletes unwanted records.

 

To begin a merge:

  1. Click People in the upper right corner of the page. The People page opens.
  2. Search for the person by entering the person's information into the search fields. You can search by Birth date and/or by name. There are two ways to search by name:
  • Enter a Last name or First name to match the name exactly.
  • Enter an incomplete Last name or First name and select the Use starts with search box to perform a search with only partial name information. The less you enter, the wider the search.

Click Search. Once the search is done, a list of names appears under the search fields.

  1. Review the list of names and find the name that you want to merge people into. This is the name that will remain in the database when the merge is done. This person is the Keeper.
  2. Click Merge in the same row as the Keeper. The page refreshes with the Keeper listed at the top of the page. Under that person's name is a list of other people you can choose from to merge with the Keeper. These people are Mergees.
  3. Select the Merge into check boxes in the rows of the Mergees.
  4. When you are done selecting Mergees, click Merge at the top of the page. The Mergees are now merged into the Keeper.

 

When the merge is done:

  • All events that used to be associated with the Mergees are now associated with the Keeper. For example, this means that if there was a CMR or a contact associated with the Mergee, it now will belong to the Keeper.
  • The Mergees are deleted and no longer visible in the application.
  • Addresses that appear on events for the Mergees remain unchanged; they are not changed with a merge.
