Creating a Place

To create a place:

  1. Click Places in the upper right corner of the page. The Manage Places page opens displaying its search options.
  2. Click Create New Place.
  3. Type the name of the Place.
  4. Select one or more types from the Types drop-down list.
  5. Enter the address information.
  6. Click Create.
