Merging Places

Duplicate place records can sometimes be created when anyone entering data fails to recognize that a place may already have been entered into the database. The merge places feature allows you to conveniently combine duplicate records.

Note: Use the merge feature with caution. It is a powerful tool that combines records for places and permanently changes addresses.

 

To begin a place merge:

  1. Click Places in the upper right corner of the page. The Manage Places page opens displaying its search options.
  2. Enter the name or part of the name of the place you want to search for.
  3. If you are sure about what type of place it is, select it from the Participation Type list. If you are not sure, leave Participation Type blank.
  4. Click Search. A list of places appears below the search fields.
  5. Review the list of places and find the place that you want to merge places into. This is the place that will remain in the database when the merge is done. This place is the Keeper.
  6. Click Merge in the same row as the Keeper. The page refreshes with the Keeper listed at the top of the page. Under the Keeper's place name is a list of other places you can choose from to merge with the Keeper. These places are Mergees.
  7. Select the Merge into check boxes in the rows of the Mergees.
  8. When you are done selecting Mergees, click Merge at the top of the page. The Mergees are now merged into the Keeper.

When the merge is done:

  • All events that used to be associated with the Mergees are now associated with the Keeper. For example, this means that if there was an event associated with the Mergee, it now will belong to the Keeper.
  • The Mergees are deleted and no longer visible in the application.
  • Addresses that appear on events for the Mergees are updated to the new address.
