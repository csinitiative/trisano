Editing a Place

The edit place function allows you to clean up the place records in your database, without having to go through the various pages in the application. To edit a place:

  1. Click Places in the upper right corner of the page. The Manage Places page opens displaying its search options.
  2. Enter the name or part of the name of the place you want to search for.
  3. If you are sure about what type of place it is, select it from the Participation Type list. If you are not sure, leave Participation Type blank.
  4. Click Search. A list of places appears below the search fields.
  5. Locate the place you want to edit and click Edit in that place's row. A page opens allowing you to edit the address for this place.
  6. Make any necessary changes and click Update to save your changes. The application displays a message telling you that the place was successfully updated.
  7. Click Back to Places to return to the Manage Places page.

Changes made to addresses in the Manage Places area apply to any other area where that place, such as events or place exposures, has been used.
