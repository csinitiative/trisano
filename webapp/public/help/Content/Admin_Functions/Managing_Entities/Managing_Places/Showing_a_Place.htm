Showing a Place

To show a place:

  1. Click Places in the upper right corner of the page. The Place page opens.
  2. Enter the name or part of the name of the place you want to search for.
  3. If you are sure about what type of place it is, select it from the Participation Type list. If you are not sure, leave Participation Type blank.
  4. Click Search. A list of places appears below the search fields.
  5. Locate the place you want to edit and click Edit in that place's row. The edit page for the selected place opens.
  6. Click Show. The page displays the name and address information for the place. Switch to edit mode by clicking Edit if you want to edit the information for this place.
