Adding a Value Set

A value set is a collection of answers to a question. For example, many questions throughout CSI TriSano™ make use of the collection of answers Yes, No, and Unknown. This collection of answers is referred to as a value set. Defining value sets makes it possible to reuse a collection of answers again for another question, without having to define the set of answers all over again.

 

To add a value set to a question:

  1. Click Add value set on the same row as the question it belongs with. The Value Set entry area of the page opens.
  2. Enter a name for the value set in the Value Set Name field.
  3. Click Create. This adds the Value Set to the question area of the page. Once the value set has been created, you can add values, edit values, or delete it.
  4. To add values to the value set, click Add value. This opens up an area of the screen that allows you to add the values.
  5. In the Value Name field, type the value you want to appear when a user views the possible answers to the question. Value names are the actual values available when a user completes the field on the form. For example, if the possible answers to a question were Yes, No, and Unknown, one of the values would be Yes, so in the Value Name field, you would enter Yes.
  6. In the Code field, enter the code that you want stored in the database when a user chooses this value. For the value you just entered above, you would enter Yes.
  7. To add the value, click Create. The page is updated.
  8. To finish defining the value set comprised of Yes, No, and Unknown, repeat steps 4 through 7 for No and Unknown.

The following table describes commands that are available in the Value Set section of the page once you have added a value set to a question.

 

Option Description
Inactivate Make the answer inactive for this question. It will not show up as a possible answer when the form is used with an event.
Edit Allows you to edit the value.
Trash can icon Deletes the value.
