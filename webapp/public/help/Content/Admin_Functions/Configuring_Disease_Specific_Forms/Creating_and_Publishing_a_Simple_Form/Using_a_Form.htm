Using a Form

To use the form once it has been created:

  1. Create a new CMR.
  2. Select the Clinical tab.
  3. Select a disease from the Disease list on the Clinical tab.
  4. Click Save & Continue. The application checks for any published forms that are associated with this disease and jurisdiction. If it finds one, it adds that form to this CMR.
  5. Select the Investigation tab. If there are any forms associated with the disease you selected, the form name is listed on the left side of the tab. You may see questions for that form on this tab, which is the default location for questions. In addition, questions from the form may also be added to other tabs.

Note: Complete Form Builder instructions are available on our subscription area.
