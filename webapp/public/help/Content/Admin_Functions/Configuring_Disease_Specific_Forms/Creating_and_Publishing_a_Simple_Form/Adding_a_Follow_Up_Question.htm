Adding a Follow Up Question

A follow up question allows an investigator to ask an additional question if the answer to a previous question meets specific, defined criteria.

For example, an investigator has selected Yes to the question, Does the patient have a rash? Knowing that the patient has a rash makes you want to get more information from the patient about the date the rash started. In other words, if the investigator selects the Yes radio button, indicating that the patient has a rash, the investigator should ask a follow up question to learn the date when it started.

Follow up questions are made up of two parts: the condition that must be met and the new follow up question. These two parts together are referred to as the follow up container.

Defining the Condition

To define the follow up question:

  1. Click Add follow up in the same row as the question. The Add follow up area of the page expands.
  2. Enter the condition in the Condition field. For the Rash example above, you would enter Yes in the Condition field.
  3. Click Create. A new line appears at the bottom of the Default View section of the page that says Follow Up, Condition: Yes. The follow up condition is indented slightly and is lighter in color to indicate it is tied to the previous question.

Follow up questions allow you to perform the following functions:

Function Description
Add question to follow up container Allows you to add questions that appear when the user meets the condition of the follow up. You can add more than one question. Follow up questions can also have their own follow up questions.
Edit Allows you to edit the follow up condition.
Trash can icon Deletes the follow up question and its contents.

Defining the Question

To add a question to this follow up container:

  1. Click Add question to follow up container. The Add new question area of the page expands and is the same as for adding a question to a form.
  2. Enter the text for the question in the Question text field.
  3. Enter a short name in the Short name field.
  4. Enter some instructions to the user in the Help text field.
  5. Select a data type for the answer in the Data type field.
  6. Click Create.
  7. The Default View section of the page displays the new question.
