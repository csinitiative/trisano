Manually Removing a Form from an Event

To manually remove a form from an event:

  1. Click Add/Remove forms for this event.







  2. The page that opens contains form names that are available for this event. The page has two sections:
  • Forms in use – can be removed from the event
  • Forms available for use – can be added to the event

  1. To remove a form, if there are any listed, select the check box on the right side of the page and click Remove Forms.
  2. A dialog box asks you to confirm that you want to delete the form. This means that all the data associated with that form will be lost.
  3. Click OK. The form is removed.
