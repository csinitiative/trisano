Creating a Simple Form

To create the simplest of forms takes very few steps. Begin entering some basic information:

  1. Click Admin in the upper right corner of the page.
  2. Click Manage Forms under Disease Specific Forms.
  3. Click Create new form in the upper right corner of the page. The Create Form page opens.

Entering Information

On the Create Form page, enter the following information:

  1. In the Name field, enter the name of the form. Use the name from the printed version of the form, if there is one. Use a name that is useful to you.
  2. In the Short Name field, enter a brief name for the form. Make sure that it is still recognizable as being connected with the name you entered in the Name field.
  3. Select an event type from the Show when event type is list to specify the event types the form will be used with.
  4. Select a disease from the Show when disease is list to specify the diseases the form will be used with.
  5. Select a jurisdiction or select All Jurisdictions from the Show when jurisdiction is list to specify the jurisdictions the form will be used in.
  6. Enter a description in the Description field. Enter a description that will be useful to users.

Click Create to Finish

After you enter the information as described above, click Create. The Details page opens displaying the message at the top: Form was successfully created. The Details page contains basic information about the form. The Event Type column indicated that this form is Not Published. This means that the form is not yet available for use. You must publish the form to make it available.
