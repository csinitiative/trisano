Publishing a Form

Publishing is the act of making a form available for use in CSI TriSano™. While you are still developing a form, it is unavailable to users. Once you are done creating it, you can publish it so that it can be used to gather critical data.

After you finish creating a form it is still in the Not Published state. Review how the page looks after you finish creating the form. In the Event Type column, it shows that the form is Not Published.

 

 

To publish the form, click Publish. The application returns you to the main Forms page, and displays a message telling you that the form was successfully published. The form is now available for use.
