How to Use this Guide

 

 

 

 

 

 

Purpose of this Guide

The CSI TriSano™ Administrator Guide provides information on how to use CSI TriSano Enterprise Edition Administrator Functions. This guide explains:

  • Configure the System
  • Manage Entities
  • Export Event Information
  • Configure Disease Specific Forms

Audience

This guide is intended for users at the administrative level who are fairly computer savvy and require a minimum of instruction to perform configuration tasks.It also assumes that you are familiar with and have used the CSI TriSano™ application and are familiar with its features. For more information on the features of CSI TriSano™ refer to the online help.

Assumptions

This guide assumes that you have access to CSI TriSano™ and that you are logged in as an administrator so that you can access Form Builder and create and update forms.

Formatting Conventions

This guide uses the following conventions:

  • Bold text indicates user action as in the following example:
    Type 0, then press Enter for each of the remaining fields.
  • Italic text indicates a new word or a word that requires some emphasis as in the following example:
    Before completing a CMR, we strongly recommend you choose a disease.
  • Important notes and tips are set off from the text and presented as in the following example:
    Note: You must save the new form before you can publish it.
