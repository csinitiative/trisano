How to Use this Guide

 

 

 

 

 

 

Purpose of this Guide

The CSI TriSano™ User Guide provides information on how to use CSI TriSano Enterprise Edition. This guide explains:

  • Confidential Morbidity Reports and how to create and use them
  • The different types of events and how to create and edit them
  • How to search for people and places within the application
  • How to use electronic lab reporting
  • The basics of analysis, visualization, and reporting capabilities

Audience

This guide is intended for general users who use a computer daily and are familiar with the basic and custom computer operations. It assumes an introductory knowledge of disease management but no knowledge of the CSI TriSano™ application.

Assumptions

This guide assumes that you have access to CSI TriSano™ and that you are logged in. If you do not know how to log in or have access issues, contact your IT staff.

Formatting Conventions

This guide uses the following conventions:

  • Bold text indicates user action as in the following example:
    Type 0, then press Enter for each of the remaining fields.
  • Italic text indicates a new word or a word that requires some emphasis as in the following example:
    Before completing a CMR, we strongly recommend you choose a disease.
  • Important notes and tips are set off from the text and presented as in the following example:
    Note: You must save the new form before you can publish it.
