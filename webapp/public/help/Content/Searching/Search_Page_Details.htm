Search Page Details

The CSI TriSano™ Search page allows you to search for people and events using many flexible methods. You can search for data based on any combination of the following:

  • name
  • demographic information
  • clinical data
  • event information
  • epidemiological and reporting data

The following is an image of the Search page. Different sections of the page let you establish criteria for the search. The Search page initially displays only the Name criteria section. You can show or hide other sections of the page by clicking Show|Hide All Other Criteria.

 

 

Showing/Hiding Screen Sections

You can show or hide each screen section, depending on your search needs. Click the Show | Hide red text for each section to toggle the display of that section.
