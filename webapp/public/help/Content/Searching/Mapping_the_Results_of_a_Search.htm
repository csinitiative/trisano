Mapping the Results of a Search

After you have performed a search, you may want to see those results on a map. Seeing the results on a map may be helpful in determining patterns of infection or location.

 

To map the results:

  1. Perform a search using the criteria that you need.
  2. When the search is done, the results are displayed at the bottom of the search page.
  3. Review the results. Below the search results is a link called Show Map of Results. If the link does not appear, it means that none of the events listed in the search results contain map coordinates, i.e., the Longitude and Latitude fields on the Demographic tab for these events are either empty or need to be updated.
  4. Click Show Map of Results. A map is displayed at the bottom of the page.

Viewing Mapped Results

The results map shows you a small tag for each of the events listed in the search results. Listed events that are missing address information, will not appear on the map.

The message, Some results need to be updated with map coordinates sometimes appears next to the Show map of results link. You will see this message when one or more of the search results have blank Longitude and Latitude fields. To fix this situation, you must update each patient or contact individually. To update the coordinates:

  1. In the search results area of the page, click the Record Number of one of the events whose Demographic tab is missing address information. You can predict which ones are missing address information by inspecting the City column of the search results. If the City column is empty, it probably means that this patient's address coordinates need to be updated.
  2. When the page for the events opens, click Edit.
  3. Locate the Address fields and make sure they are as complete as possible.
  4. Click Look Up Address. The application checks the address and if there is sufficient address information, displays the verified (sometimes corrected) address below the Look Up Address link. If there isn't enough address information, the application tells you so. In this case, you will not be able to include this patient or contact in the mapped results.
  5. Click Update Coordinates. The Longitude and Latitude fields will be updated if the address fields contain enough information.
  6. Click Save & Continue.
