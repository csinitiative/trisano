Managing Test Types and LOINC Codes

LOINC Codes—Logical Observation Identifiers Names and Codes—were developed to make it easier to exchange and pool clinical results, outcomes management, and research by providing a set of standard codes and names to identify laboratory and other clinical observations. The codes are made up of a collection of numbers that are not easy to remember. CSI TriSano™ allows you to define Test Types and connect them to LOINC Codes to simplify their use.

Many codes and test types are already defined when you install CSI TriSano™. But since the codes are constantly changing, with new ones being added all the time, you may receive lab results that use a code that is not in your database. You can use the LOINC Code List to define a new code and then use the Test Type List to connect a name to one or more codes. The Test Type name will appear on the lab report when it is assigned to an event on the Laboratory tab.

Adding a LOINC Code

To begin connecting a LOINC Code with a Test Type, you first want to define the LOINC Code:

  1. Click Admin in the upper right corner of the page.
  2. Under Lab Management, select Manage LOINC Codes.
  3. Click Create New LOINC Code.
  4. Enter the LOINC Code in the LOINC Code field.
  5. Enter the name for the test in the Test name field.
  6. Select a Common test type from the drop-down.
  7. Select a scale for the test from the Scale drop-down.
  8. Select an organism from the Organism drop-down list.
  9. Click Create. The application reports that the code was successfully created.

Connecting the LOINC Code to a Test Type

To define a Test Type and assign LOINC Codes to it:

  1. Click Admin in the upper right corner of the page.
  2. Under Lab Management, select Manage Common Test Types.
  3. Click Create New Common Test Type.
  4. Enter a name in the Common name field and click Create. The application reports that the name was successfully created.
  5. Click LOINC Codes in the row of the name you just created. Search fields open allowing you to search for a LOINC Code by code or by name or by both. Complete the search fields as follows and click Search.
  • To search by LOINC code, enter a code in the LOINC Code field.
  • When you search by LOINC code, it compares the start of the code. So 114 returns 114-0 and 11432-9.
  • To search by test name, enter a name in the Test name field.
    • Searching for a test name compares any part of the test name. So a search for 'culture' returns any LOINC code with a test name that contains the word 'culture'.
  • To search by both, use both fields as already described.
    • Searching for both returns only LOINC codes that match criteria entered both search fields.

Any LOINC codes already associated with the current Test Type are excluded from the search results.
A list of found codes is displayed under the search field.

  1. Select the codes you want to connect to this Test Type and click Add. The LOINC Codes are now associated with this Test Type.
