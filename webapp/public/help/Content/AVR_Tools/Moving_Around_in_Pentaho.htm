Moving Around in Pentaho™

You should rely upon the documentation provided by Pentaho™ for specific details on how it works. Check their Web site for documentation: pentaho.com. There are also basic instructions available on our subscription area.

Use one of the three main buttons along the center of the screen to create:

  • a new report - that allows you to create ad hoc reports
  • an analysis view - launches the analysis tool
  • a new dashboard - allows you to create a dashboard
