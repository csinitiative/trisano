Editing Keyboard Shortcuts

Keyboard shortcuts are key strokes assigned to a defined set of CSI TriSano™ functions. You can use the keyboard shortcuts in place of pointing and clicking with the mouse.

 

To specify keyboard shortcuts:

  1. Click the Settings link in the upper right corner of the page. The User Settings page opens.
  2. Click Edit Keyboard Shortcuts under Navigation. The Configure Shortcuts page opens listing the functions of the application that allow shortcuts.
  3. Select one of the application's functions from the list. A text field opens that allows you to enter a keyboard shortcut.
  4. Type the shortcut for the function. Shortcuts can consist of the CTRL, SHIFT, or ALT keys used alone or in combination, along with another key stroke, pressed at the same time.
  5. Edit one or many of the keyboard shortcuts. After you edit a shortcut, its text appears in green.
  6. Click Update Shortcuts. After you click Update Shortcuts, the text for the shortcuts appears in gray, like the rest of the text on this screen.
