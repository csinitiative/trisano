Changing the Main Page Filter

Tasks appear on the main page of CSI TriSano™ based on filter specifications that you make when you click Change Filter. To view and modify the current filter specification:

 

  1. Click Change Filter on the main page. The page expands to display a list of filtering criteria.
  2. Set new filter specifications. The table below describes the filter settings.
  3. Click Update to save the new filter settings. The filter area closes and the task list is updated to reflect the changes you just made.

Field Description
Show tasks for the last number of days Enter a number in the text box to specify how many days in the past the filter should look to display tasks. Leaving the field blank selects all overdue tasks.
Show tasks for the next number of days Enter a number in the text box to specify how many days in the future the filter should look to display tasks. Leaving the field blank selects all pending tasks.
Disease filter Select the diseases you would like to see listed on the main page task list. When you select one or more diseases, the filter selects tasks for only the checked diseases. Leaving all the check boxes blank selects tasks for all diseases.
Users filter Select the users whose tasks you would like to see listed on the main page task list. When you select one or more users, the filter selects tasks for only the checked users. Leaving all the check boxes blank selects tasks for all users.
Jurisdictions filter Select the jurisdictions whose tasks you would like to see listed on the main page task list. When you select one or more jurisdictions, the filter selects tasks for only the checked jurisdictions. Leaving all the check boxes blank selects tasks for all jurisdictions.
Status filter Select the task statuses you would like to see listed on the main page task list. When you select one or more statuses, the filter selects tasks for only the checked statuses. Leaving all the check boxes blank selects tasks for all statuses.
