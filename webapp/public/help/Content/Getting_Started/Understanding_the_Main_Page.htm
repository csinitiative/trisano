Understanding the Main Page

When you launch CSI TriSano™, the main page, also known as the Dashboard, appears as in the illustration below. The links in the upper right corner of the page are always available and allow you to navigate to the different functional areas of the application.

 

The table of tasks in the main body of the page displays:

  • due date
  • name of the task
  • disease
  • description
  • category
  • priority
  • assigned to
  • status

 

The tasks are listed based on filter specifications. If there are no tasks defined, then none will be listed on the Dashboard. Click Change Filter to view and modify the current filter specification. You can create tasks while a CMR is in show mode.

 

Click on a column head to sort the list by that heading.

 

You can move quickly to an event or a task by clicking Edit Event or Edit Task. In addition, you can change the status of a task using the Status drop-down. Tasks that are complete appear with a gray background.

 

Move your mouse over the illustration below and click the different screen areas when a hand cursor appears to learn about the different areas of the page. Click the words in red text to read about the features available from the events list.
