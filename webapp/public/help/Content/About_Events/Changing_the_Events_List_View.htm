Changing the Events List View

When you first view the events list, it lists all morbidity events and contact events that have been routed. You can use the Change View command to specify what is displayed on the events list:

  1. Click Change View. The page expands to display view options.
  2. Specify settings as described in the table below.
  3. Click the Change View button at the bottom of the Change View area. The events list changes according to the specifications you have made. An event must meet all the conditions you specify in order to be included in the new view.
  4. Click Close at the bottom of the change view page area to collapse the change view options.

 

Field Description
Events Select the event status from the list. You can select more than one using the CTRL key. Only events with the selected statuses will be listed.
Queues Select a queue from the list. You can select more than one using the CTRL key. Only selected queues will be listed.
Investigators Select an investigator from the list. The list of investigators includes only those users with the investigator role in the jurisdictions you are allowed to view. You can select more than one using the CTRL key. Only events for the selected investigators will be listed.
Diseases Select any of the listed diseases. Only events for the diseases you select will be listed. You can select more than one disease by holding down the Ctrl key and clicking additional selections.
Items per page Select a number from the list to determine how many events will be displayed on a page.
Do not show deleted records Select this option to prevent deleted records from appearing on the list.
Set as default view Select this option to make the new changes the default view for the events list.
