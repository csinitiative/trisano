Understanding Events

Events record information that helps state and local health departments track and report surveillance data. They allow you to record information about:

  • a patient who shows symptoms of a nationally notifiable disease or other diseases of public health interest (morbidity event)
  • other people who the patient came in contact with (contact event)
  • an encounter the patient had with a health care provider (encounter event)
  • a place where contact occurred (place exposure)

You create contact events, encounter events, and place exposures as part of a morbidity event.

Links on the events list, which displays morbidity events and contact events that have been routed to you, allow you to perform the following tasks:

  • Show an event
  • Edit an event
  • Print the event
  • Delete an event
  • Change the view of the events list
  • Create a new morbidity event
  • Export all events to a CSV file
  • Route an event to a local health department

Links on the individual events page allow you to perform the following tasks:

  • Add a task to an event
  • Add an attachment to an event
  • Export a single event to a CSV file
  • Create a new morbidity event from this event

Links on an individual contact event page in Show mode allow you to perform the following tasks:

  • Add a task to an event
  • Add an attachment to an event
  • Promote a contact event to a morbidity event
