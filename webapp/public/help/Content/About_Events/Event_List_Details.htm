Event List Details

The events list displays all morbidity events and contact events that have been routed. To view the events list, click Events in the upper right corner of the page. The list opens, displaying one event per row. Each event listed shows:

  • patient or contact name
  • disease
  • jurisdiction
  • status
  • event type
  • date the event was recorded

When you first view the list of events, it is sorted by the date the event was saved. Click a column head to sort by that column.

Click on the commands in red text to perform an action on an event.
