Deleting an Encounter Event

To delete an encounter event that is part of a morbidity event:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the morbidity event that contains the encounter event you want to delete.
  3. Click Edit. The page for the morbidity event opens.
  4. Select the Encounter tab.
  5. Locate the Encounter you want to delete. Select the Remove check box.
  6. Click Save & Continue. The encounter event is still listed, but is no longer available for edit.
