Exporting All Events

You can export the list of events to CSV for review and analysis from the following locations in the application:

  • after performing an event search you can click the Export All to CSV link at the top of the list of search results
  • at the top of the events list you can click the Export to CSV link

To export events:

  1. Click Export All to CSV (or Export to CSV). The page expands to display some export options.
  2. Select from among the following to include in the export:
  • Contacts
  • Place Exposures
  • Lab Results
  • Treatments
  1. Select Use Short Column Names to have the export use the short names defined on the CSV Export Columns page.
  2. In the Codes and Descriptions area, select the radio button next to each field to specify if you want to export the code or the description for the listed field. Codes and descriptions are visible on the Manage System Codes page.
  3. Click Export. The File Download dialog opens.
  4. Click Open to view the export or click Save to save it to disk.

 

Notes:

  • When you run the export from the search results page, if the search delivers multiple pages of results, only the page you are viewing is exported. In addition, only built-in fields are exported, unless you choose to export data for only one disease. This means that none of the disease-specific data is exported unless you have chosen only the one disease.
  • When you run the export from the events list, only built-in fields are exported.
  • If you want the export to contain disease-specific data, export only one event at a time.
