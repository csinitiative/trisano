Adding an Attachment to a Morbidity or Contact Event

An attachment is a file that adds information to the data about the morbidity or contact event. It can be a piece of correspondence, lab results, a letter, an x-ray, or many other types of files. To add an attachment:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the morbidity or contact event you want to add the attachment to.
  3. Click Show.
  4. Click Add attachment. The New Attachment page opens.
  5. Enter a file name or click Browse to locate the file on disk.
  6. Select a Category from the list.
  7. Click Create. The attachment is saved with the morbidity or contact event record and is listed at the bottom of the page. It will also be listed on the Notes tab. In both locations, the attachment will be listed with an Open link if it is in a recognizable format. Click Open to open the attachment in its native application.
