Exporting a Single Event

You can export an event to CSV for review and analysis. To export an event:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the morbidity event you want to export.
  3. Click Show for this event.
  4. Click Export to CSV. The page expands to display export options.
  5. Select from among the following to include in the export:
  • Contacts
  • Place Exposures
  • Lab Results
  • Treatments
  1. Select Use Short Column Names to have the export use the short names that are defined on the CSV Export Columns page, which is managed by a system administrator.
  2. In the Codes and Descriptions area, select the radio button next to each field to specify if you want to export the code or the description for the listed field. Codes and descriptions are visible on the Manage System Codes page, which is managed by a system administrator.
  3. Click Export. The File Download dialog opens.
  4. Click Open to view the export or click Save to save it to disk.
