Editing an Event

To edit a morbidity or a contact event that has been routed to you:

  1. Click Events in the upper right corner of the page. The events list opens.
  2. Review the list and locate the event you want to edit.
  3. Click Edit. The page displays the tabbed information for this event in edit mode.
  4. Make any necessary changes and click Save & Exit to save your changes and return to the events list.

To edit a contact event that has not been routed:

  1. Click Events in the upper right corner of the page. The events list opens.
  2. Review the list and locate the morbidity event that contains the contact event you want to edit.
  3. Click Edit. The page displays the tabbed information for the morbidity event in edit mode.
  4. Select the Contacts tab.
  5. Review the list of contacts and locate the contact you want to edit.
  6. Click Edit contact.
  7. Make any necessary changes and click Save & Exit to save your changes and return to the events list.

To edit an encounter event (part of a morbidity event):

  1. Click Events in the upper right corner of the page. The events list opens.
  2. Review the list and locate the morbidity event that contains the encounter event you want to edit.
  3. Click Edit. The page displays the tabbed information for this event in edit mode.
  4. Select the Encounters tab.
  5. Click Edit encounter event.
  6. Make any necessary changes and click Save & Exit to save your changes and return to the events list.

To edit an existing place exposure (part of a morbidity event):

  1. Click Events in the upper right corner of the page. The events list opens.
  2. Review the list and locate the morbidity event that contains the place exposure you want to edit.
  3. Click Edit. The page displays the tabbed information for this event in edit mode.
  4. Select the Epidemiological tab.
  5. Click Edit place details.
  6. Make any necessary changes and click Save & Exit to save your changes and return to the events list.
