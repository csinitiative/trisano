Showing a Morbidity or Contact Event

To show a morbidity or contact event:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the event you want to view.
  3. Click Show.

The page displays the tabbed information for this event in show mode, meaning that the data is not available for editing. Switch to edit mode by clicking Edit if you want to edit the event data.

 

In show mode, morbidity events have these additional commands that don't appear in edit mode:

  • Add Task - allows you to add a task.
  • Add Attachment - allows you to add an attachment.
  • Export to CSV - allows you export just this single event to a CSV file.
  • Create a new event from this one - allows you to create a new event from the one you are viewing.
  • Events - returns you to the Event List page.

In show mode, contact events have these additional commands that don't appear in edit mode:

  • Add Task - allows you to add a task.
  • Add Attachment - allows you to add an attachment.
  • Promote to CMR - promotes the contact event to a morbidity event.
