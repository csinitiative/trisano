Creating a New Event from an Existing One

While there is a morbidity event (CMR) open in show mode, you can create a new CMR using the displayed event as the source for some of the information for the new one. Copying is a good practice in that it helps to prevent duplicate patients in the database. To create a new event from an existing one:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the morbidity event you want to use as the source.
  3. Click Show for this event.
  4. Click Create a new event from this one.
  5. The page expands to display the following two options for copying:
  1. Click Create & Edit Shallow Copy to create a new CMR that copies the demographic information from the original one.
  2. Click Create and Edit Deep Copy to create a new CMR that copies the demographic information from the original one, along with any of the following selections:
    • Clinical information (without disease)
    • Labs and lab results
    • Reporting information
    • Disease forms and answers
    • Notes

    Select any of the options to copy them to the new CMR.

After you click one of the Create buttons, the new CMR is created with the options you selected and opens on the page.
