Routing a Morbidity or Contact Event

Routing describes forwarding the morbidity event information (or case) for a patient or contact to a different local health department. The ability to route a case depends on your user role. If you do not have rights to route cases, the rows for events will show the text Routing disabled in gray text.

 

To route a case:

  1. Click Events in the upper right corner of the page.
  2. Review the list and locate the morbidity or contact event you want to route.
  3. Click Route to Local Health Depts. The page expands to display routing options.
  4. Specify settings as described in the table below.
  5. Click Route Event to route this event or click Close to close the Routing area without taking action.

 

Field Description
Investigating Jurisdiction Select the investigating jurisdiction from the list. You can select only one.
Also grant access to: Select as many additional jurisdictions as you like.
Brief note Enter a short note in the text field.
