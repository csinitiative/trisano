== Welcome to TriSano Core

TriSano Core is the Rails based web application which supports the collection, management,
and analysis of data for public health experts. For more information please visit 
http://www.trisano.com/overview

== Getting Started

The best source of information regarding TriSano is available on the TriSano community wiki
at https://wiki.csinitiative.com/display/tri/Home. It includes system requirements, architecture,
installation, and development documentation.
