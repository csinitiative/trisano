Note to the project owner

This is the Releases page. You can use this page to communicate about the releases planned for this project. Project members will want to know when each release is due, its current status, the artifacts that are expected to be resolved in it, and other information.

One good way to track releases is to create a freeform page for each release. Clicking the name of a given release shows that release's home page.

By default, this note is only visible to project owners. To hide it from everyone, click Edit this page and set the availability of the first HTML component to None.
