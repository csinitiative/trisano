Note to the project owner

This is the Project tools page. This is where project participants will get to the software development tools they will use on this project.

You can remove tools that you don't need by deleting the table rows for those tools. You can also add new rows with links to other Web-based tools in your environment.

By default, this note is only visible to project owners. To hide it from everyone, click Edit this page and set the availability of the first HTML component to None.
