This is the Working system work products page. You can use this page to communicate and store working system related documentation (release notes, operational procedures, training, etc.). This is also a great place to capture customer feedback before they get converted to user stories or defect reports in Project Tracker.

By default, this note is only visible to project owners. To hide it from everyone, click Edit this subpage and set the availability of the first HTML component to None.
