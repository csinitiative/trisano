Note to the project owner

This is the Project FAQ (Frequently Asked Questions) page. The best FAQs are written by project members themselves as they participate in the project over time. It's a good idea to seed this page yourself with some initial questions and answers. Try to think of questions that a real project participant might ask, such as "Where are the project meetings held?"

By default, this note is only visible to project owners. To hide it from everyone, click Edit this subpage and set the availability of the first HTML component to None.
