CollabNet's agile process comes with a set of ready-to-use assets:

  • Project pages. A project owner can use these to manage project information, share documentation, provide views into project artifacts, and much more.

  • Project Tracker artifacts. User stories, tasks, and defect reports help plan and track the scope of releases and iterations through the lifecycle of the project.

  • Project Tracker queries. Project members can quickly query artifacts to get started on their own assignments, or to get a deep dive into project scope.
