CollabNet's agile process helps you manage globally distributed development projects effectively. Your CollabNet project is a hub for communication and collaboration, giving globally distributed team members a common and unified vision.

Using CollabNet's project workspace, project members can work as one team throughout the lifecycle of a software project, regardless of their location. The project's evolving scope, progress, and quality are visible to key stakeholders at all times.  This is especially critical in a project following the agile process as timely communication on plans and status is key to the success of the project.

Total adaptability

CollabNet project templates can be completely adapted to your organizational and project needs. Use this template as the starting point for developing your own internal process.

If you come up with a process that you would like to share with other projects, your host administrator can save this project as a template that others can select during project creation.

Note: If your project follows a simple process or a Unified Process (such as the Rational Unified Process™), you can start with a template designed to support that process. For information about available templates, see openCollabNet.

Close control

Timely, relevant information helps business owners and other stakeholders align distributed software development practices with desired business objectives. CollabNet provides a topical view of the project that supports the information needs of all the project's stakeholders, regardless of their technical ability. You can switch between a "tool-centric" view and a "process-centric" view, whichever fits your requirements and work patterns.

CollabNet's rich set of editing capabilities provide an easy way to display, organize, and find information about your project. You get a combination of page definitions, document templates, and Project Tracker components (such as artifact types, attributes, attribute options, and queries), all of which can be tailored to your project's unique needs.
