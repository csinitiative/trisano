Before you begin contributing to the project, you will want to familiarize yourself with the project:

  • Source code
  • Directory and file organization
  • Metadata
  • Revision history

You have two options for displaying code and metadata from the project repository:

  • Display existing source code with the online version control tool browser of Subversion. Source code repositories are accessed at the top level of the Projects area.
  • Check out source code with a Subversion client. Subversion clients and plug-ins allow you to access Subversion from your current development environment. For information about Subversion clients and plugins, see openCollabNet.
